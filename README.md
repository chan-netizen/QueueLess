# 🎫 QueueLess – Virtual Queue Management App

Skip the wait. Join queues virtually from your phone and arrive just in time.

---

## 🚀 Live Demo

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-name.streamlit.app)

## 📌 Features

- **User Mode** – Join a queue at any business and see live position & wait time
- **Business Dashboard** – Call next customers, track queue length, view analytics
- **Real-time Updates** – Queue position refreshes instantly
- **Multi-business Support** – Bank, Clinic, Restaurant, Salon, Government Office

---

## 🛠️ Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/queueless-app.git
cd queueless-app

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

---

## ☁️ Deploy to Streamlit Cloud (Free)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **"New app"**
4. Select your repo → branch: `main` → file: `app.py`
5. Click **Deploy** 🎉
